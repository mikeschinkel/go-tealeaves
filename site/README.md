# go-tealeaves Documentation Site

Astro + Starlight documentation site for [go-tealeaves](https://github.com/mikeschinkel/go-tealeaves).

## Development

```bash
bun install
bun run dev       # http://localhost:4321
bun run build     # Production build to ./dist/
bun run preview   # Preview production build locally
```

## Deployment

Deployed automatically to GitHub Pages via `.github/workflows/deploy-docs.yml` on push to `main`.

Live site: https://mikeschinkel.github.io/go-tealeaves/
