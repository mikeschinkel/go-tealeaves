## Install

```bash
go get github.com/mikeschinkel/go-tealeaves/teamodal
```

Progress indicator for long-running operations with optional background and cancel support. Displays "Generating *Title*..." with Cancel and optional Background hints.

## Quick Example

{/* verified: site/examples/progress-dialog/compile_test.go:11 */}
```go
modal := teamodal.NewProgressModel(&teamodal.ProgressModelArgs{
    Title:             "Commit Message",
    BackgroundEnabled: true,
})

// Open it
m.modal = m.modal.Open()

// In View()
if m.modal.IsOpen() {
    return m.modal.OverlayModal(backgroundView)
}

// Handle responses
case teamodal.ProgressCancelledMsg:
    // User pressed Esc
case teamodal.ProgressBackgroundMsg:
    // User pressed 'b' to send to background
```

## Features

- Displays "Generating *Title*..." with inline cancel/background key hints
- Cancel with Esc key
- Optional background mode (press `b` to send operation to background)
- ANSI-aware overlay compositing via `OverlayModal()`
- Non-nil `tea.Cmd` message consumption signal (see [Modal Message Consumption](/go-tealeaves/guides/modal-consumption/))
- Auto-centering on screen based on `SetSize()` dimensions

## API Reference

### Constructor

| Function | Signature | Description |
|----------|-----------|-------------|
| `NewProgressModel` | `NewProgressModel(args *ProgressModelArgs) ProgressModel` | Create a progress indicator dialog |
| `DefaultProgressModelKeyMap` | `DefaultProgressModelKeyMap() ProgressModelKeyMap` | Returns default key bindings (Esc = cancel, b/B = background) |

### Key Types

| Type | Description |
|------|-------------|
| `ProgressModel` | Progress indicator with cancel/background support |
| `ProgressModelArgs` | Configuration: `Title`, `BackgroundEnabled`, `ScreenWidth`, `ScreenHeight` |
| `ProgressModelKeyMap` | Key bindings — `Cancel` (Esc) and `Background` (b/B); accessible via `ProgressModel.Keys` |

### ProgressModelKeyMap Fields

| Field | Default keys | Description |
|-------|-------------|-------------|
| `Cancel` | `esc` | Cancel the operation |
| `Background` | `b`, `B` | Send the operation to background |

### Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `Open` | `Open() ProgressModel` | Open the modal; pre-calculates overlay position |
| `Close` | `Close() ProgressModel` | Close the modal |
| `IsOpen` | `IsOpen() bool` | Whether the modal is currently open |
| `SetSize` | `SetSize(width, height int) ProgressModel` | Set screen dimensions for centering |
| `SetTitle` | `SetTitle(title string) ProgressModel` | Update the title at runtime |
| `SetBackgroundEnabled` | `SetBackgroundEnabled(enabled bool) ProgressModel` | Enable or disable the Background option at runtime |
| `OverlayModal` | `OverlayModal(background string) (view string)` | Render modal centered over a background view |
| `Update` | `Update(msg tea.Msg) (ProgressModel, tea.Cmd)` | Handle keyboard and window-size messages; returns typed `ProgressModel`, not `tea.Model` |

### Messages

| Message | When sent |
|---------|-----------|
| `ProgressBackgroundMsg` | `b` pressed in progress modal |
| `ProgressCancelledMsg` | Esc pressed in progress modal |

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Esc` | Cancel operation |
| `b` | Send to background (when enabled) |

## Known Limitations

<Aside type="caution">
- Requires screen size to be set via `SetSize()` for correct centering.
- `ProgressModel.Update()` returns `(ProgressModel, tea.Cmd)`, not `(tea.Model, tea.Cmd)` — store the result in a typed field, not a `tea.Model` interface.
- The progress display is static text; there is no animated spinner.
- `ProgressModel` has no `WithTheme()` — apply styling directly or call `DefaultBorderStyle()` / `DefaultTitleStyle()` from the `teamodal` package.
</Aside>

### Error Variables

| Variable | Description |
|----------|-------------|
| `ErrModal` | Sentinel error for the `teamodal` package |
| `ErrCancelled` | Returned when the modal is cancelled |
| `ErrInvalidBounds` | Returned on invalid position bounds |
