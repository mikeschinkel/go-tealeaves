Tea Leaves is a Go component library for [Bubble Tea](https://github.com/charmbracelet/bubbletea) TUI applications. It provides 17 composable model types across 12 component packages, plus 5 foundation packages used by those components.

## Views

Full-featured view models that manage data display and keyboard navigation.

<CardGrid>
  <Card title="Grid View" icon="table">

  </Card>
  <Card title="Tree View" icon="list-format">

  </Card>
  <Card title="Drilldown View" icon="right-arrow">

  <Card title="Status Bar" icon="bars">

## System

Foundational packages used by the components above.

| Package | Slug | Description |
|---------|------|-------------|
| `teahelp` | [Help Visor](/go-tealeaves/components/help-visor/) | Scrollable key-binding help overlay |
| `teapane` | [Pane Widgets](/go-tealeaves/components/pane-widgets/) | Ready-made `tealayout`-compatible pane widgets |
| `teautils` | [Key Registry](/go-tealeaves/components/key-registry/) | Centralized key binding registry for status bars and help overlays |
| `teautils` | [Theming](/go-tealeaves/components/theming/) | Consistent color palette and style system |
| `teautils` | [Positioning](/go-tealeaves/components/positioning/) | Terminal-aware overlay positioning utilities |
| `teacolor` | [Color Constants](/go-tealeaves/components/color-constants/) | Named ANSI color constants |
| `teahilite` | [Syntax Highlighting](/go-tealeaves/components/syntax-highlighting/) | Syntax highlighting via Chroma |
