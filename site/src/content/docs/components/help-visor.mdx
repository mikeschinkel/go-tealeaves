`HelpVisorModel` is a full Bubble Tea component that renders a paginated, categorized keyboard shortcuts overlay. It supports multiple pages with left/right navigation, category ordering, and a "visor" appearance (open bottom border). The host application provides categorized key bindings via `Open()`; `teahelp` handles rendering, pagination, and dismissal.

## Install

```bash
go get github.com/mikeschinkel/go-tealeaves/teahelp
```

## Quick example

{/* verified: site/examples/help-visor/compile_test.go:15 */}
```go
// Create once, embed in your root model
help := teahelp.NewHelpVisorModel()
help = help.SetSize(termWidth, termHeight)

// Open with key bindings grouped by category
keysByCategory := map[string][]teautils.KeyMeta{
    "Navigation": {
        {Binding: upKey,   HelpText: "Move up"},
        {Binding: downKey, HelpText: "Move down"},
    },
    "Actions": {
        {Binding: enterKey, HelpText: "Select"},
        {Binding: escKey,   HelpText: "Cancel"},
    },
}
help = help.Open(keysByCategory)

// In Update() ΓÇõ route to help first when it's open
var cmd tea.Cmd
help, cmd = help.Update(msg)
// Handle ClosedMsg when user dismisses
case teahelp.ClosedMsg:
    // Help was closed, return focus to the application

// In View() ΓÇõ place at the top of the screen
if help.IsOpen() {
    return help.View().String() + "\n" + mainContent
}
return mainContent
```

## Features

- Categorized key binding display with configurable category order
- Automatic pagination: key entries split across pages based on terminal height
- Left/right arrow navigation between pages, digit keys (1ΓÇô9) for direct page jump
- Open-bottom "visor" border style
- Configurable title, category, key, and description styles via `WithContentStyle()`
- Configurable chrome styles (border, footer) via `WithStyles()`
- `ClosedMsg` emitted when user dismisses

## API Reference

### Constructors

| Function | Description |
|----------|-------------|
| `NewHelpVisorModel()` | Creates a closed help visor with default styles and key bindings |

### Lifecycle methods

| Method | Description |
|--------|-------------|
| `Open(keysByCategory)` | Opens the visor with the given key bindings and builds pages |
| `Close()` | Closes the visor and clears state |
| `SetSize(width, height)` | Updates terminal dimensions; rebuilds pages if currently open |
| `IsOpen()` | Returns whether the visor is currently displayed |
| `Page()` | Returns the current page index (0-based) |
| `TotalPages()` | Returns the total number of pages |

### Wither methods

| Method | Description |
|--------|-------------|

| `WithStyles(styles HelpVisorStyles)` | Override chrome styles (border, footer key/label) |

### Bubble Tea interface

| Method | Description |
|--------|-------------|
| `Init()` | Returns nil (no initial command) |

## Key map (HelpVisorKeyMap)

`HelpVisorKeyMap` configures the keys that control the help visor. Override via `WithKeys()`:

| Field | Default keys | Description |
|-------|-------------|-------------|
| `Close` | `esc`, `?` | Close the visor and emit `ClosedMsg` |
| `PrevPage` | `ΓÁÉ` | Navigate to the previous page |
| `NextPage` | `ΓÁÈ` | Navigate to the next page |

{/* verified: site/examples/help-visor/compile_test.go:39 */}
```go
km := teahelp.DefaultHelpVisorKeyMap()
km.Close = key.NewBinding(key.WithKeys("esc"), key.WithHelp("esc", "close help"))
help = help.WithKeys(km)
```

## Content style (HelpVisorStyle)

The content style controls the rendered key binding entries. Configure via `WithContentStyle()`:

{/* verified: site/examples/help-visor/compile_test.go:47 */}
```go
help = help.WithContentStyle(teautils.HelpVisorStyle{
    TitleStyle:    lipgloss.NewStyle().Bold(true),
    CategoryStyle: lipgloss.NewStyle().Foreground(teacolor.Teal),
    KeyStyle:      lipgloss.NewStyle().Foreground(teacolor.BrightWhite),
    DescStyle:     lipgloss.NewStyle().Foreground(teacolor.SlateGray),
    KeyColumnGap:  4,
    CategoryOrder: []string{"Navigation", "Actions", "System"},
})
```

| Field | Type | Description |
|-------|------|-------------|
| `TitleStyle` | `lipgloss.Style` | Style for the "Keyboard Shortcuts" title line |
| `CategoryStyle` | `lipgloss.Style` | Style for category heading lines |
| `KeyStyle` | `lipgloss.Style` | Style for key display labels |
| `DescStyle` | `lipgloss.Style` | Style for description text |
| `KeyColumnGap` | `int` | Gap (in spaces) between key and description columns |
| `CategoryOrder` | `[]string` | Preferred order for categories; unlisted categories follow alphabetically |
