> **Module:** `teadiffr` · **Type:** `TUIRenderer` · **Install:** `go get github.com/mikeschinkel/go-tealeaves/teadiffr`

`teadiffr` renders condensed file diffs in a terminal-friendly format. It provides a `DiffRenderer` interface and a default `TUIRenderer` that uses lipgloss for colored `+`/`-` line output.

## Install

```bash
go get github.com/mikeschinkel/go-tealeaves/teadiffr
```

## Key Types

- **`FileDiff`** — Represents a single file's diff with status, blocks, and metadata
- **`CondensedBlock`** — A contiguous range of added/deleted/context lines
- **`DiffRenderer`** — Interface for rendering diff output
- **`TUIRenderer`** — Default implementation with lipgloss styling
