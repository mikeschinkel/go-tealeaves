## Directory Layout

{/* non-compilable: directory tree diagram */}
```
go-tealeaves/
├── teacolor/      # Named color constants (standalone module)
├── teacrumbs/     # Breadcrumb navigation
├── teadiff/       # Diff viewer (split + condensed)
├── teafields/     # Dropdown selection
├── teagrid/       # Data grid
├── teaguide/      # Workflow guidance overlay
├── teahelp/       # Help system utilities
├── teahilite/     # Syntax highlighting
├── tealayout/     # Declarative layout engine
├── teamodal/      # Modal dialogs + list + choice + progress
├── teanotify/     # Toast notifications
├── teapane/       # Pane widgets (StyledPane, ScrollPane, PlainPane)
├── teastatus/     # Status bar
├── teatext/       # Text selection + clipboard
├── teatree/       # Tree view + drilldown
├── teautils/      # Key registry, help visor, theming, positioning
├── examples/      # Per-component examples
├── site/          # Documentation site (Astro/Starlight)
└── go.work        # Workspace for local development
```

## Versioning

Each module is tagged independently using the pattern `<module>/v0.X.Y`:

{/* non-compilable: version tag examples */}
```
teafields/v0.3.0
teagrid/v0.3.0
teamodal/v0.3.0
...
```

This means upgrading one component never forces upgrades to others.

## Dependency Graph

Most components depend only on `teautils` for shared helpers. `teatree`'s `DrillDownModel` also depends on `teafields` for its dropdown UI. No other cross-component dependencies exist.

{/* non-compilable: dependency graph diagram */}
```
teacolor ───► (none — only lipgloss)
teafields ──► teautils
teagrid ────► teautils
teamodal ───► teautils
teanotify ──► teautils
teatree ────► teautils, teafields
teatext ────► teautils
teastatus ──► teautils
teadiff ────► teautils
teacrumbs ──► (none — only bubbletea + lipgloss)
tealayout ──► (none — only lipgloss)
teapane ────► tealayout
teaguide ───► teautils
```
