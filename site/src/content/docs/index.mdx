## Components

<CardGrid>
	<Card title="teadd" icon="list-format">
		Dropdown selection with intelligent positioning. Displays above or below the anchor based on available terminal space.
	</Card>
	<Card title="teamodal" icon="document">
		Modal dialogs for confirmations, alerts, and choice/list selection. Supports button focus, mouse clicks, and custom key bindings.
	</Card>
	<Card title="teatree" icon="seti:folder">
		Generic tree view with expand/collapse, keyboard navigation, viewport scrolling, and pluggable node providers.
	</Card>
	<Card title="teastatus" icon="bars">
		Two-zone status bar with key-action menu items on the left and text indicators on the right.
	</Card>
	<Card title="teatextsel" icon="pencil">
		Text area with selection (Shift+Arrow) and clipboard operations (Ctrl+C/X/V). Supports single-line mode.
	</Card>
	<Card title="teautils" icon="setting">
		Key registry, help visor, and positioning utilities shared across Bubble Tea applications.
	</Card>
	<Card title="teadep" icon="random">
		Interactive dependency path viewer for exploring module dependency graphs.
	</Card>
	<Card title="teanotify" icon="information">
		Notification overlays for transient status messages and alerts.
	</Card>
	<Card title="teagrid" icon="seti:db">
		Grid/table component for tabular data display.
	</Card>
</CardGrid>

## Installation

Each package is an independent Go module. Install the ones you need:

```bash
go get github.com/mikeschinkel/go-tealeaves/teadd
go get github.com/mikeschinkel/go-tealeaves/teamodal
go get github.com/mikeschinkel/go-tealeaves/teatree
```

There is no root module — do not `go get` the repository root.
