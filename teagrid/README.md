# teagrid

A Bubble Tea component for interactive, customizable tables. Fork of [bubble-table](https://github.com/evertras/bubble-table) by Brandon Fulljames.

## Installation

```bash
go get github.com/mikeschinkel/go-tealeaves/teagrid
```
